Python Package

simplemachinelearning

Summary

In this project I created a python Machine Learning package. The objective of it is to load in any data set within a CSV file, explore, pre-process and then build a supervised learning model writting as little code. as possible. Both classification and regression models can be built. The GitHub code , example use cases and PyPi website are also linked.
