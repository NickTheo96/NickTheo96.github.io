Space Invaders Type Game

Summary

For this project, I developed a space invaders type game. It is at an early stage where enemies are generated and a player in the shape of a rocket is able to move around the screen using keyboard controls. It was built using the Chilli Framework. Details of the framework can be found here. The next step in the game would be to allow the player to file missiles at the enemy. Please refer to the GitHub section to view the code and instructions on how to compile it.
